# crystal-vix

Crystal binding and library to use VMware VIX interface for controlling ESX\ESXi hosts and guests  

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  crystal-vix:
    github: bararchy/crystal-vix
```
